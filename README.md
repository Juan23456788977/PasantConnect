## 📋 Descripción

**PasantConnect** es un sistema web completo para la administración de pasantías, diseñado para centralizar y automatizar la gestión de pasantes, empresas tutoras y moderadores académicos en una sola plataforma.

Desarrollado durante mis pasantías en **Akdemia** como solución real a un problema real: la gestión manual y caótica del proceso de pasantías estudiantiles.

---

## ✨ Características Principales

- 🔐 **Sistema de autenticación multi-rol** (Pasante, Empresa, Moderador)
- 📊 **Dashboard interactivo** con métricas en tiempo real
- 🏢 **Registro y gestión de empresas** tutoras
- 👤 **Gestión completa de pasantes** con seguimiento de progreso
- 📄 **Generación automática de reportes** en PDF
- 📱 **Diseño responsive** adaptado a móviles y escritorio
