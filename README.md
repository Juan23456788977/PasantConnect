## 🌍 Overview / Descripción General

**(EN)** **PasantConnect** was my graduation thesis project for I.T.A.T. It is a full-featured management system for interns, companies, and supervisors. It includes hour tracking, real-time chat, photo uploads, dark mode, and a comprehensive admin panel. Security features include **bcrypt** password hashing and role enforcement via MySQL `enum` validation at every endpoint.

**(ES)** **PasantConnect** es mi proyecto de tesis de grado para el I.T.A.T. Es un sistema completo de gestión de pasantes, empresas y tutores. Incluye control de horas, chat en tiempo real, subida de imágenes, modo oscuro y panel de administración. La seguridad incluye encriptación **bcrypt** y control de roles estricto validado en cada endpoint contra la base de datos.
